Программа позволяет проверить сложность пароля. 
Реализована на основе паттерна Template Method.
Подходящий пароль должен соответствовать следующим требованиям: 

1. Длина от 8 до 32-х символов
2. Два и более символов алфавита в верхнем регистре
3. Два и более символов алфавита в нижнем регистре
4. Две и более цифры
5. Запрет на 3 одинаковых символа, идущих подряд
6. Отсутствие пробела
7. Хотя бы 1 специальный символ из перечня:
 ! @ # $ % ^ & * ( ) - _ = + \ | [ ] { } ; : / ? . > <
